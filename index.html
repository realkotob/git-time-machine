Powerful Yet Simple

A focused interface to inspect and recover reachable local Git history, wrapped in an intuitive interface.

⌨️

Keyboard-Driven

Navigate with arrow keys, vim bindings (j/k), or Home/End. Fast and efficient workflow for developers who love the terminal.

👁️

Live Diff Preview

Press Space to see git diff --stat before restoring. Know exactly what you're getting back with detailed change previews.

🛡️

Safety First

Confirmation dialogs show the exact Git command, hard resets create backup refs first, and --list-backups shows recovery commands later.

🎨

Beautiful UI

Clean, modern terminal interface with syntax highlighting. Makes reflog browsing actually enjoyable.

⚡

Lightning Fast

Built in Rust for maximum performance. Instant startup, smooth scrolling, responsive interactions.

🔍

Smart Filtering

Filter by message, hash, author, or relative time. Press y to copy the selected hash, or ? for contextual help.

When You Need It Most

Real scenarios where git-time-machine helps you choose a recovery point

💥

Recover from Bad Rebase

Rebased and need an earlier local state? Browse your reflog visually, preview the target, then choose hard reset, soft reset, or detached checkout.

🔙

Undo Accidental Reset

Did git reset --hard and need to go back? Time machine shows recent local reflog states, previews the target, creates a backup ref before another hard reset, and can list those backup refs later.

🌿

Find Deleted Branches

Accidentally deleted a branch? If the tip is still in your local reflog, navigate through history, find the commit, and recreate the branch manually.

🕵️

Investigate History

Curious what happened in your repo? Browse through all operations, see when branches were created, when merges happened, and understand your project's evolution.
